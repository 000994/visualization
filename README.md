# Michelin Guide — Global Star Restaurants (2018–2019)

一个交互式数据可视化项目，展示全球 **695 家米其林星级餐厅**（2018–2019）的分布。支持地图、图表、多维度筛选与收藏功能。

---

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可（**无需服务器**，纯静态页面）。推荐使用最新版 Chrome / Edge / Firefox。

> 如果遇到地图瓦片不显示，检查网络是否能访问 `cartocdn.com`。

---

## 🎮 操作指南

### 1️⃣ 左侧面板

| 区域 | 功能 |
|---|---|
| **统计卡片** | 显示总数 / ★★★ / ★★ / ★ 的数量。**点击可快速筛选**对应星级。 |
| **星级筛选** | 选择 1★ / 2★ / 3★，可多选 |
| **价格筛选** | 选择价格范围 $ → $$$$$，可多选 |
| **搜索框** | 按餐厅名称模糊搜索 |
| **↺ Clear All** | 一键重置所有筛选条件 |

### 2️⃣ 图表筛选（菜系 & 地区）

- **📊 Cuisine Distribution**（横向柱状图）：点击某个菜系条，**过滤出该菜系的所有餐厅**
- **🍩 Region Distribution**（圆环图）：点击某个地区扇区，**过滤出该地区的所有餐厅**
- 再次点击同一项可**取消过滤**
- 被选中的条/扇区保持高亮（其他变淡）

> 🎯 点击图表后，**地图自动缩放**到该菜系/地区的餐厅范围，列表同步更新。

### 3️⃣ 交互式地图

- **拖拽 + 滚轮缩放**浏览全球
- 标记按星级着色（★ 青铜、★★ 银灰、★★★ 金色）
- 密集区域自动聚合为**聚类标记**（显示数量）
- 地图右下角有**图例**

### 4️⃣ 餐厅列表

- 列表项显示：收藏按钮 ♡ → 星级 → 餐厅名 + 菜系·城市 → 价格
- **点击列表项** → 地图飞入 + 打开详情面板
- **鼠标悬浮列表** → 地图对应标记高亮
- **排序下拉**：按名称 / 星级 / 价格排序

### 5️⃣ 详情面板

点击列表中任意餐厅后会从右侧滑入，显示：

- 餐厅全名 + 星级（彩色）
- 菜系、价格、城市、地区、年份
- **相似菜系推荐**（同菜系的其他餐厅，点击快速跳转）
- → **View on MICHELIN Guide** 在新窗口打开官网

### 6️⃣ ❤️ 收藏功能

| 操作 | 说明 |
|---|---|
| 点击列表中的 **♡** 按钮 | 添加/取消收藏 |
| 点击 Header 的 **♡ Favorites** | 进入「仅看收藏」模式（按钮变金色） |
| 再次点击退出 | 回到完整列表 |
| 收藏保存在浏览器本地 | 关闭页面后仍在 |

### 7️⃣ 深色模式

点击 Header 中的 **🌓** 按钮切换深色/浅色/系统跟随模式。

### 8️⃣ 键盘快捷键

| 按键 | 功能 |
|---|---|
| ↑ ↓ | 列表导航 |
| Enter | 打开当前高亮的餐厅 |
| Esc | 关闭详情面板 |
| ? | 显示/隐藏快捷键帮助 |

### 9️⃣ 年份筛选

Header 上的 **All Years / 2019 / 2018** 按钮可切换年份。

## 🏗️ 项目结构

```
├── index.html          # 主页面
├── css/
│   └── style.css       # 全部样式（深色模式变量）
├── js/
│   ├── data.js         # 695 家餐厅数据
│   ├── main.js         # 状态管理、筛选、列表、详情、收藏
│   ├── map.js          # Leaflet 地图 + 标记聚类
│   └── chart.js        # Chart.js 图表 + 点击筛选
└── README.md
```

## 📦 技术栈

- [Leaflet](https://leafletjs.com/) + [MarkerCluster](https://github.com/Leaflet/Leaflet.markercluster) — 交互地图
- [Chart.js](https://www.chartjs.org/) — 菜系/地区统计图
- 原生 JavaScript（ES5 兼容） — 无构建工具
- CSS 自定义属性 — 深色/浅色主题

## 📊 数据来源

数据爬取自 [Michelin Guide 官网](https://guide.michelin.com/)，包含 2018 和 2019 两个年份的星级餐厅信息。
